Paramètres généraux :

  • Nom du projet : définit le nom de l'archive qui sera téléchargeable via le bouton "Générer"
  • Nom de la table : permet de générer un fichier texte avec du code SQL qui crée une table avec ce nom et enregistre les fichiers de sortie avec ce nom
  • Nombres de lignes : définit le nombre de lignes de données à générer
  • Graine de génération : permet de choisir un nombre qui, si vous le ré-utilisez vous donnera les mêmes données lors de la génération
  • Formats de sortie : permet de choisir le ou les formats de sortie pour les données

Champs de données :

  • Nom : nom de la donnée, fournit une en-tête avec ce nom dans le CSV et/ou un champ de ce nom avec des valeurs dans une table en SQL

  • Types de données :

    • Dictionnaire : permet de choisir parmi une liste de dictionnaires disponibles. Les dictionnaires sont en rapport avec un thème.
    • Numérique : permet de générer de manière aléatoire un nombre entre une valeur minimum et une maximum
    • Formule : calcule des valeur selon une formule du modèle a * x [ + - * / ou % pour modulo] b où a, x et b peuvent être des noms de colonne ou des nombres.
  • Null : Nombre de lignes de données qui seront non définies, cette valeur peut être saisie en pourcentage ou en valeur absolue
  • Codage : système de correspondances pour faire correspondre des valeurs à des codes, comme par exemple "1;Homme" associera le code 1 à la valeur "Homme"

Les différents boutons :

  • Remettre à zéro : recharge la page
  • Ajouter/Supprimer une colonne : permet l'ajout/ la suppression d'un champ de données
  • Ajouter/Supprimer table : ajoute une table en dernière position ou supprime la dernière table
  • Importer un fichier XML ou SQL : va permettre de précharger la page de génération via une création de table SQL ou un fichier de paramétrage XML
  • Générer : lance le script de génération
