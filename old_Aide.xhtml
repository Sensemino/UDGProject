Page D'aide de l'interface

Les champs généraux :

  • Nom du projet : définie le nom de l'archive que vous pourrez télécharger à la fin du script
  • Nom de la table : permet de générer un SQL créant une table avec ce nom, enregistre les fichiers de sortie avec ce nom
  • Nombres de lignes : définit le nombre de ligne de données que vous voullez
  • Graine de génération : vous permet de choisir un nombre qui, si vous le ré-utilisez vous donnera les mêmes données lors de la génération
  • Formats de sortie : vous permet de choisir un ou plusieurs formats de données à la fin du script

Les champs de données :

  • Nom : nom de la donnée, vous permet d'avoir une en-tête avec ce nom dans le CSV ou de rentrer les valeurs dans une table en SQL

  • Type de données :

    • Dictionnaire : va vous permettre de choisir parmis une liste de dictionnaires celui qui vous convient. Les dictionnaires sont en rapport avec un thème.
    • Numérique : va vous permettre de générer de manière aléatoire un nombre entre une valeur minimum et une maximum
    • Formule : une formule sous la form a * x [ + - * / ou % pour modulo] b où a, x et b peuvent être des noms de colonne ou des nombres.
  • Null : Nombre de ligne de données qui seront non définies, peut être en % ou en valeur brut
  • codage : système de codage pour faire correspondre des valeurs à un nombre exemple "1;homme" associera tout les 1 à homme

Les différents boutons :

  • Remettre à zéro : recharge la page
  • Ajouter/Supprimer une colonne : permet l'ajout/ la suppression d'un champ de données
  • Ajouter/Supprimer table : ajoute une table en dernière position ou supprime la dernière table
  • Importer un fichier XML ou SQL : va permettre de précharger la page de génération via une création de table SQL ou un fichier de paramétrage XML
  • Générer : lance le script de génération
